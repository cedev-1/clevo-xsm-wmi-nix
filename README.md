# clevo-xsm-wmi-nix

A Nix package for the Clevo XSM WMI kernel driver.

The driver enables control of Clevo laptops’ special function keys via the WMI interface.

Source code is taken from the upstream repository:
https://github.com/quitesimpleorg/clevo-xsm-wmi-preserved

In my case it's for this laptop model : Clevo NJ50MU - [My Nix Configuration (Thorfinn)](https://github.com/cedev-1/nix-configuration)
